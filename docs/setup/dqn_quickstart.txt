╚════════════════════════════════════════════════════════════════╝

1. INSTALL DEPENDENCIES
   pip install torch numpy robocode-tank-royale

2. RUN THE BOT
   
   Option A (Recommended):
   python3 -m bots.python.dqn.runtime.run_bot
   
   Option B:
   bash run.sh
   
   Option C (with custom settings):
   python3 -m bots.python.dqn.runtime.run_bot --batch-size 64 --learning-rate 1e-4

3. MONITOR TRAINING
   Watch the console output for episode progress
   
   Example output:
   [DQNBot] Episode 1 | eps_end=0.900 | buffer=0
   [DQNBot] Episode 1 finished | won=True | reward=2.15 | win_rate=1.000

4. OUTPUTS
   • bots/python/dqn/checkpoints/dqn_weights.pt → Saved model (auto-loads on restart)
   • bots/python/dqn/logs/dqn_training_log.jsonl → Episode metrics

5. CUSTOMIZE (all optional)
   --learning-rate RATE       Default: 1e-4
   --gamma DISCOUNT           Default: 0.99
   --eps-start VALUES         Default: 0.9
   --eps-end values           Default: 0.05
   --eps-decay-steps STEPS    Default: 2500
   --tau SOFT_UPDATE          Default: 0.005
   --batch-size SIZE          Default: 128
   --memory-capacity SIZE     Default: 10000
   --weights-path PATH        Default: bots/python/dqn/checkpoints/dqn_weights.pt
   --log-path PATH            Default: bots/python/dqn/logs/dqn_training_log.jsonl
